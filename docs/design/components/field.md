# Field

**Figma**: [Field Component](https://www.figma.com/design/SAFZI8SsNAwUMo8vZm9DwD/AdaptMap-Designsystem?node-id=842-49181&p=f&m=dev)

**Location**: Form field wrapper components

## Description

Container component for form inputs with labels, supporting both vertical and horizontal layouts.

## Layout Variations

1. **Vertical Field**
   - Label positioned directly above input element
   - Radio button and checkbox options listed vertically

2. **Horizontal Field**
   - Labels for text inputs, select, textarea, and slider remain above elements
   - Radio button and checkbox options arranged horizontally on same line

## Supported Input Types

- Text Input
- Select (Dropdown)
- Radio Button Group
- Textarea
- Checkbox Group
- Slider

## Design Variables

- **Label Typography**: Regular weight, smaller than titles, dark gray/black
- **Input/Placeholder Text**: Regular weight, similar size to labels, lighter gray for placeholders
- **Borders**: Thin, light gray border on all interactive form elements
- **Border Radius**: Small, consistent rounding across all input elements
- **Spacing**: Consistent vertical spacing between field components

## Colors

- **Interactive Elements Background**: White (`#FFFFFF`)
- **Input Text**: Dark gray/black
- **Placeholder Text**: Light gray
- **Selected Radio/Checkbox**: Solid black fill for active state
- **Slider Track**: Light gray
- **Slider Thumb**: Black
