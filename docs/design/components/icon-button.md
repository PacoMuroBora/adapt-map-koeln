# Icon Button

**Figma**: [Icon Button Component](https://www.figma.com/design/SAFZI8SsNAwUMo8vZm9DwD/AdaptMap-Designsystem?node-id=842-44443&p=f&m=dev)

**Location**: `src/components/ui/button.tsx` (icon variant)

## Variants

Same variants as Button (Primary, Secondary, Outline, Ghost, Ghost Muted, Destructive) but icon-only.

## Sizes

- **Regular**: Standard square icon button
- **Large**: Larger square icon button
- **Small**: Smaller square icon button
- **Mini**: Smallest square icon button

## Shapes

- **Default**: Square with rounded corners
- **Round**: Perfect circle

## States

- **Default**: Variant-specific background and icon color
- **Hover & Active**: Background darkens/lightens, icon color may change
- **Focus**: Variant background with thicker, lighter ring outline
- **Disabled**: Light gray background, darker gray icon, reduced opacity

## Icon Colors

- **Primary**: White icon on lime green background
- **Secondary**: Lime green icon on white background with light gray border
- **Outline**: White icon on dark gray/black background with white border
- **Ghost**: Light gray icon on transparent background
- **Ghost Muted**: Very faint light gray icon on transparent background
- **Destructive**: White icon on orange/red background

## Usage Examples

Icons supported: `+` (plus), pencil/edit, filter, kebab menu (three dots), trash can
